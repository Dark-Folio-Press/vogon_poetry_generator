# Vogon Poetry Generator

> *“Oh freddled gruntbuggly…”*

This is a poetry engine inspired by the gloriously hostile bureaucratic anti-poetry of the Vogons from The Hitchhiker's Guide to the Galaxy.

Built for:

* Douglas Adams fans
* AI enthusiasts
* ML tinkerers
* procedural literature nerds
* anyone who enjoys absurdism

# Editorial Notice I
No actual books or texts of Douglass Adam's or any author was used in this project.
The putrid unpleasantness of the generated text is achieved completely through a collection of hand built lexicon's plucked from publicly available Vogon lore, entwined with layers of distortion, corruption, and bureaucratic interuptions.

# What Is This?

This project combines:

* Large Language Models
* procedural text corruption
* lexical mutation
* syntax distortion
* streaming interfaces
* literary evaluation systems

to generate poetry that is:

* pompous
* administratively swollen
* emotionally hazardous
* cosmically unpleasant

The system intentionally mutates otherwise coherent poetry through:

* bureaucratic rewrites
* committee interruptions
* phonetic ugliness
* procedural sludge injection
* escalating corruption layers
